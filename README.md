PSF Docs
========

The Python Software Foundation Supplements are a staging area for notes,
answers to frequently asked questions or other material
before potentially being transfered to
the main [Python Software Foundation] [PSF] documentation.

The published documents are hosted by on [Read The Docs] [RTD].

The principal author of this material is [Jesse Noller] [jnoller].
To contribute you may contact Jesse via [email]
or send a [pull request] [GitHub Pull].

  [GitHub Pull]: http://help.github.com/send-pull-requests/
  [PSF]: http://www.python.org/psf/
  [RTD]: http://readthedocs.org/docs/psf-docs
  [jnoller]: http://jessenoller.com/
  [email]: jnoller@python.org
